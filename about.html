О проекте TaskFlow

TaskFlow — это консольное приложение для управления задачами, реализованное полностью с нуля на языке Python.

Проект выполнен в рамках варианта 2.2 «Практическая реализация технологии “с нуля”».

Технологии:

  • Python 3
  • JSON для хранения данных
  • colorama для цветного интерфейса
  • ООП (классы Task, TaskManager, Storage)

Основные возможности:

  • Добавление, удаление, поиск задач
  • Отметка выполнения
  • Фильтрация по статусу
  • Постоянное хранение данных в tasks.json
