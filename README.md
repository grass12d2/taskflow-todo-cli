# TaskFlow (To Do List CLI)

Простое консольное приложение для управления задачами на Python с хранением в JSON.

## Функциональность

- Добавление / удаление задач
- Отметка выполнения
- Фильтрация по статусу
- Поиск по ключевому слову
- Постоянное хранение данных
- Красивый цветной интерфейс

## Как запустить

```bash
pip install -r requirements.txt
python main.py
```
