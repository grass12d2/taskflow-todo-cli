**Простое, но мощное консольное приложение для управления задачами**, разработанное полностью **с нуля** на Python в рамках проектной практики (вариант 2.2).

---

## 📋 О проекте

**TaskFlow** — это удобный менеджер задач из командной строки с постоянным хранением данных в JSON-файле. Приложение имеет красивый цветной интерфейс и удобное меню.

Проект выполнен в рамках **проектной (учебной) практики** по дисциплине «Проектная деятельность».

---

## ✨ Основные возможности

- ➕ Добавление новых задач с описанием
- ❌ Удаление задач
- ✅ Отметка выполнения / снятие отметки
- 📋 Фильтрация задач (все / выполненные / невыполненные)
- 🔍 Поиск по ключевому слову
- 💾 Автоматическое сохранение данных в `tasks.json`
- 🎨 Красивый цветной интерфейс (colorama)
- 📊 Статистика прогресса в реальном времени

---

## 🛠 Технологии

- **Python 3.11+**
- JSON для хранения данных
- OOP (объектно-ориентированное программирование)
- colorama — цветной вывод в консоль
- Модульная структура проекта

---

## 🚀 Как запустить

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ваш_ник/taskflow-todo-cli.git
cd taskflow-todo-cli

# 2. Установите зависимости
python3 -m pip install -r requirements.txt

# 3. Запустите приложение
python3 main.py
```
